****Pour exécuter le projet et le connecter à une base de données MongoDB, voici les étapes et les commandes nécessaires que vous pouvez utiliser :

Installer Django : pip install django
Installer Bootstrap : pip install django-bootstrap5
Installer l'outil d'Encryption : pip install bcrypt
Pour la génération de PDF : pip install reportlab
Installer PyMongo : pip install pymongo

****Si vous n'êtes pas dans le répertoire "dash", vous pouvez utiliser la commande suivante dans votre terminal pour vous y rendre : cd dash.

***Ensuite, lancez le serveur en utilisant la command : python manage.py runserver

****il ya  un fichier nommé data.py pour la connexion entre MongoDB et l'application. avec nom de database > **mongoconn**
